- `service-worker.js`

- `.nojekyll`

## Mise en ligne sur GitHub Pages

### Option 1 : Nouveau dépôt
1. Crée un nouveau dépôt GitHub, par exemple `brian-crea`.
2. Dézippe ce dossier sur ton ordinateur.
3. Envoie **tout le contenu** du dossier à la racine du dépôt.
4. Va dans **Settings > Pages**.
5. Dans **Build and deployment** :
   - **Source** : `Deploy from a branch`
   - **Branch** : `main`
   - **Folder** : `/ (root)`
6. Enregistre.

Ton site sera ensuite disponible sur l’URL GitHub Pages du dépôt.

### Option 2 : Déjà un dépôt existant
1. Ouvre ton dépôt.
2. Remplace les anciens fichiers par ceux de ce dossier.
3. Vérifie que `index.html` est bien à la racine.
4. Vérifie que GitHub Pages pointe bien sur la branche `main` et le dossier `/ (root)`.
